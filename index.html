The programme allows the user to view a summary report of all stock available, view the total stock in each length category, further visualising it via a bar chart. Weekly Expense Tracker

With the use of a text file, the programme is able to gather details of the users spending habits throughout the week and categorise them by day and category which is decided on through the users personal description for each expense entry. The user can then use a menu interface to generate various structured reports of their spending, and access the total spend in each category via a table report and a bar chart, and the total spend that week. This can give vital insight on the users spending habits allowing them to manage and monitor their spending habits.
